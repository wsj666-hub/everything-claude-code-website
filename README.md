# Everything Claude Code Website

Everything Claude Code 项目的介绍网站，展示 agents、skills、commands 等功能的详细信息。

## 功能特性

- 13 个专业 Agents 详细介绍
- 45+ Skills 最佳实践
- 32+ Commands 快速命令
- 点击卡片查看详细功能和使用说明
- 响应式设计，支持移动端

## 技术栈

- 纯 HTML/CSS/JavaScript
- 无框架依赖
- 响应式设计

## 在线预览

直接打开 `index.html` 文件即可在浏览器中预览。
